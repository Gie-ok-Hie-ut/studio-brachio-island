## Contributions
- GAN 기반 고해상도 landscape image synthesis를 위해 coarse-to-fine conditional pipeline과 model architecture를 설계했습니다.
- 사용자가 빠르게 제어할 수 있으면서도 세밀한 이미지를 생성할 수 있는 controllable generation workflow를 구축했습니다.

<!-- en -->
Conditional pipelines and model architectures for high-resolution landscape image synthesis and controllable image generation.

## Contributions
- Designed a coarse-to-fine conditional pipeline and model architecture for high-resolution landscape image synthesis using GANs.
- Enabled fast, user-controlled, and highly detailed visual generation workflows.
- Built an automated synthetic face dataset pipeline that generates datasets from single face images through latent manipulation in GANs.
