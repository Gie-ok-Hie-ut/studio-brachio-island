Diffusion 기반 이미지 생성 AI SaaS 플랫폼(VARCO)과 실제 서비스용 creative tooling을 위한 모델 및 기능 개발을 수행했습니다.

- Base foundational model과 corresponding adaptor 개발 연구를 수행했습니다.
- LoRA 기반 generative model fine-tuning과 multi-GPU training을 활용해 virtual try-on 기술 개발을 주도했습니다.
- Data refinement, model training, AI architecture pipeline design을 통합해 multi-view image generation 기능을 설계하고 구현했습니다.
- Lighting adaptation과 automatic coloring을 가능하게 하는 volume generation 기능 개발을 주도했습니다.

<!-- en -->
Diffusion-based image generation AI SaaS platform (VARCO) and production-facing creative tooling.

## Contributions
- Conducted research on developing base foundational models and corresponding adaptors.
- Spearheaded virtual try-on technology using LoRA-based generative model fine-tuning with multi-GPU training.
- Designed and implemented multi-view image generation features, integrating data refinement, model training, and AI architecture pipeline design.
- Led volume generation features for lighting adaptation and automatic coloring.
